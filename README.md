# Фронт React

## Установка и запуск

1. Перейдите в папку фронтенда:

```bash
cd frontend
```

2. Установите зависимости (подтянутся все необходимые пакеты, включая Vite, TypeScript и Tailwind):

```bash
npm install
```

3. Запустите проект в режиме разработки:

```bash
npm run dev
```

4. Проверьте порт запуска CORS политика бэкенда разрешает 5173 (деф порт запуска Vite).

После запуска проект будет доступен по адресу, который покажет Vite, обычно [http://localhost:5173](http://localhost:5173).
